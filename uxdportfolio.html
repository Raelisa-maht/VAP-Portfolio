UXD - Hotel room webpage and room booking webpage

Project Background

To design and develop an intuitive web user interface as a team using HTML, CSS and JavaScript for a hotel, creating 1 content intensive webpage and 1 interactive online form webpage per person.

My role and solution

In the UXD project for the Hotel Room webpage and Room Booking webpage, I was responsible for designing and developing intuitive web interfaces using HTML, CSS, and JavaScript. This involved creating a visually engaging Hotel Room webpage to showcase room information and services, as well as a user-friendly Room Booking webpage with interactive features for seamless booking. My focus was on meeting project objectives while prioritizing usability and delivering exceptional user experiences.

Hotel Room Webpage

This hotel room webpage displays the types of rooms provided in this particular hotel. Each of the rooms seen in this picture has a short description. By clicking on the see more button, it would then take you to a more detailed webpage regarding the room.

Room information webpage

This room information webpage displays a description of the room as well as the various amenities that comes alongside with the room upon booking. This webpage also allows users to check on the availability of the room upon booking. Finally, this webpage also shows you the pictures of various parts of the room.

Room Booking Webpage

This room booking webpage displays the fields users are required to fill in to book the rooms. The booking page usually includes a summary of the reservation, total cost, and any additional options (like breakfast or airport transfers). To complete the booking, users provide their personal details and payment information.

Confirmation page

Confirmation details and a receipt are provided once the booking is confirmed, offering a seamless and efficient experience for users looking to secure their accommodation.
